# Sourcecode of the Official FruchtLabor TeamSpeak Bot

## installation

```
npm i
```

## required files

### .env

```
NODE_ENV="development"
```

oder

```
NODE_ENV="production"
```

### .env.dev / .env.prod

```
TEAMSPEAK_IP=""
TEAMSPEAK_PORT=0
TEAMSPEAKQUERY_USERNAME=""
TEAMSPEAKQUERY_PASSWORD=""
TEAMSPEAK_NICKNAME=""

MONGODB_CONNECT=""
MONGODB_DBNAME=""
```
